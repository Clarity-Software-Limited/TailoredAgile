# TailoredAgile

TailoredAgile is a storefront-style site where teams can mix and match software delivery methodologies, practices, and patterns to build a bespoke SDLC framework.

## Features

- Left-side catalog of selectable agile/engineering practices
- Cart workflow with add, remove, and quantity controls
- Running cart total and item count
- Checkout summary modal to review your custom framework bundle
- Responsive layout for desktop and mobile

## Run Locally

Because this is a static site, you can run it with any local HTTP server.

Example:

```bash
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000`
